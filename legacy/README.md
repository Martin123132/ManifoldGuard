# Legacy demos

This folder contains historical exploratory scripts and notebook-style demo
material from earlier ManifoldGuard and pre-ManifoldGuard work.

These files are kept for provenance, not as the recommended public API. Prefer
the current package, CLI, docs, and examples at the repository root.
