## Release notes

This release adds focused offline support for:

- EXP24 conditional, unit/range, comparison, and retrieval-sensitive probes
- EXP25 hard-negative, section-reference, and numeric-scope probes
- deterministic offline corpus evaluation through completed EXP24 and EXP25 runs
- clearer package identity and installation guidance for offline-first use
- Dependabot guardrails for unsupported `sentence-transformers` major updates

No public API signature changes were introduced.
